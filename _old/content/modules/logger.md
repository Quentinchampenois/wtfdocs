Displays the contents of the WTF log file. The log file is located at `~/.config/wtf/log.txt`.

## Usage

To log to this file in your own modules:

{{< code lang="go" >}}
require "github.com/wtfutil/wtf/logger"
logger.Log("This is a log entry")
{{< /code >}}

## Configuration

{{< code lang="yaml" >}}
logger:
  enabled: true
  position:
    top: 5
    left: 4
    height: 2
    width: 1
  refreshInterval: 1
{{< /code >}}
