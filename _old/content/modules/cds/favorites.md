Display runs of you [CDS](https://ovh.github.io/cds/) favorites workflows.

<img class="screenshot" src="/imgs/modules/cds_favorites.png" width="520" alt="CDS Favorites screenshot" />
## Configuration

{{< code lang="yaml" >}}
cdsFavorites:
  enabled: true
  position:
    top: 0
    left: 0
    height: 2
    width: 2
  refreshInterval: 8
  apiURL: https://api.cds.localhost.local
  token: xxxxxxxxxxxx
  hideTags:
  - "git.repository"
  - "git.hash"
  - "triggered_by"
{{< /code >}}
