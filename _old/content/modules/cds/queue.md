Display current [CDS](https://ovh.github.io/cds/) queue - waiting or building jobs.

<img class="screenshot" src="/imgs/modules/cds_queue.png" width="520" alt="CDS Queue screenshot" />


## Configuration

{{< code lang="yaml" >}}
cdsQueue:
  enabled: true
  position:
    top: 0
    left: 0
    height: 3
    width: 2
  refreshInterval: 8
  apiURL: https://api.cds.localhost.local
  token: xxxxxxxxxxxx
{{< /code >}}
