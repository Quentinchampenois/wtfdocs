Runs a terminal command on a schedule.

## Configuration

{{< code lang="yaml" >}}
cmdrunner:
  args: ["-g", "batt"]
  cmd: "pmset"
  enabled: true
  position:
    top: 6
    left: 1
    height: 1
    width: 3
  refreshInterval: 30
{{< /code >}}
