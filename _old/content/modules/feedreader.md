A rudimentary RSS & Atom feed reader.

## Configuration

{{< code lang="yaml" >}}
feedreader:
  enabled: true
  feeds:
  - https://news.ycombinator.com/rss
  - https://rss.cbc.ca/lineup/topstories.xml
  feedLimit: 10
  position:
    top: 4
    left: 1
    height: 1
    width: 2
  refreshInterval: 14400
{{< /code >}}
