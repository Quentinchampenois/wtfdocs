Displays a configurable digital clock.

## Configuration

{{< code lang="yaml" >}}
digitalclock:
  color: orange
  enabled: true
  font: bigfont
  hourFormat: 12
  position:
    top: 0
    left: 1
    height: 1
    width: 1
  refreshInterval: 1
  title: "big clock"
  type: "digitalclock"
{{< /code >}}
