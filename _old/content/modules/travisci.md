Displays build information for your Travis CI account.

## Configuration

This module pulls information from api.travis-ci.com (when pro: true) or api.travis-ci.org (when pro: false)

{{< code lang="yaml" >}}
travisci:
  apiKey: "p0d13*********************************************c3"
  enabled: true
  compact: true
  limit: 8
  sort_by: "id:desc"
  position:
    top: 4
    left: 1
    height: 1
    width: 2
  pro: false
  refreshInterval: 900
{{< /code >}}
