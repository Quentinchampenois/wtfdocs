Connects to the Datadog API and displays alerting modules.

## Configuration

{{< code lang="yaml" >}}
datadog:
  apiKey: "<yourapikey>"
  applicationKey: "<yourapplicationkey>"
  enabled: true
  monitors:
    tags:
      - "team:ops"
  position:
    top: 4
    left: 3
    height: 1
    width: 2
{{< /code >}}
