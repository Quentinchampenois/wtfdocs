Displays chat messages from Gitter.

## Configuration

{{< code lang="yaml" >}}
gitter:
  apiToken: "d23*******************************************3r2"
  enabled: true
  numberOfMessages: 10
  position:
    top: 4
    left: 1
    height: 1
    width: 4
  roomUri: wtfutil/Lobby
  refreshInterval: 300
{{< /code >}}
