Displays stories from Hacker News.

## Configuration

{{< code lang="yaml" >}}
hackernews:
  enabled: true
  numberOfStories: 10
  position:
    top: 4
    left: 1
    height: 1
    width: 2
  storyType: top
  refreshInterval: 900
{{< /code >}}
