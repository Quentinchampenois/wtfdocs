Displays your current IP address information, from ipinfo.io.

**Note:** IPInfo.io has a free-plan rate limit of 1000 requests per day.

## Configuration

{{< code lang="yaml" >}}
ipinfo:
  colors:
    name: red
    value: white
  enabled: true
  position:
    top: 1
    left: 2
    height: 1
    width: 1
  refreshInterval: 150
{{< /code >}}
