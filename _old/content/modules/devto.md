Displays stories from The Practical DEV [dev.to](https://dev.to).

## Configuration

{{< code lang="yaml" >}}
devto:
  enabled: true
  numberOfArticles: 10
  position:
    top: 1
    left: 1
    height: 1
    width: 2
  contentTag: "showdev" 
  contentUsername: "victoravelar"
  contentState: "rising"
{{< /code >}}
