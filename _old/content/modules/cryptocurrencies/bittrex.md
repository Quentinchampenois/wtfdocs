Get the last 24 hour summary of cryptocurrencies market using [Bittrex](https://bittrex.com).

## Configuration

{{< code lang="yaml" >}}
bittrex:
  colors:
    base:
      name: orange
      displayName: red
    market:
      name: red
      field: white
      value: green
  enabled: true
  position:
    top: -2
    left: 2
    height: 3
    width: 1
  refreshInterval: 5
  summary:
    BTC:
      displayName: Bitcoin
      market:
      - LTC
      - ETH
{{< /code >}}
