[wego](https://github.com/schachmat/wego) is a command line tool to view
weather, from a variety of weather services

<img class="screenshot" src="/imgs/modules/cmdrunner/wego.png" alt="wego screenshot" />

## Config Example

{{< code lang="yaml" >}}
weather:
  args: ["0"]
  cmd: "wego"
  enabled: true
  type: "cmdrunner"
  position:
    top: 0
    left: 0
    height: 1
    width: 2
  refreshInterval: 100
{{< /code >}}
