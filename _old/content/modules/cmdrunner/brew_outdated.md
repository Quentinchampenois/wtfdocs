Displays a list of all the installed [Homebrew](https://brew.sh) recipes that have an update available for them.

## Config Example

{{< code lang="yaml" >}}
brew_outdated:
  args: ["outdated"]
  cmd: "brew"
  enabled: true
  position:
    top: 3
    left: 1
    width: 2
    height: 1
  type: cmdrunner
{{< /code >}}
