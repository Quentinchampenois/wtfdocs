Displays CPU and memory usage.

## Configuration

{{< code lang="yaml" >}}
resourceusage:
  cpuCombined: false
  enabled: true
  position:
    top: 1
    left: 1
    height: 1
    width: 1
  refreshInterval: 1
  showCPU: true
  showMem: true
  showSwp: true
{{< /code >}}
