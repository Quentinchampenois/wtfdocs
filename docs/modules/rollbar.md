# Rollbar

Displays item information for your rollbar account.

## Configuration

```yaml
rollbar:
  accessToken: "d23*******************************************3r2"
  enabled: true
  projectOwner: "ENCOM"
  projectName: "MCP"
  activeOnly: true
  assignedToName: "dillinger"
  count: 3
  position:
    top: 4
    left: 1
    height: 2
    width: 2
  refreshInterval: 900
```
