# Blockfolio

Display your Blockfolio crypto holdings.

## Configuration

```yaml
blockfolio:
  colors:
    name: blue
    grows: green
    drop: red
  device_token: "device token"
  displayHoldings: true
  enabled: true
  position:
    top: 3
    left: 1
    width: 1
    height: 1
  refreshInterval: 400
```
