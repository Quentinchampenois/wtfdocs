# Finnhub

Display Finnhub stocks for the configured symbols.

## Configuration

```yaml
finnhub:
  apiKey: "your-api-key"
  symbols:
    - "AAPL"
    - "MSFT"
    - "AMZN"
    - "FSLY"
  enabled: true
  position:
    top: 0
    left: 0
    height: 2
    width: 2
  refreshInterval: 5
```
