# VictorOps

Connects to the VictorOps API and determines who is on call.

## Configuration

```yaml
victorops:
  apiID: a3c5dd63
  apiKey: "p0d13*********************************************c3"
  enabled: true
  position:
    top: 0
    left: 1
    height: 2
    width: 1
  refreshInterval: 3600
  team: devops
```
