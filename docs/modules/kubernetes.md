# Kubernetes

Displays information about a Kubernetes cluster.

## Configuration

```yaml
kubernetes:
  enabled: true
  kubeconfig: "/Users/testuser/.kube/config"
  namespaces:
  - internal
  - public
  - systems
  objects:
  - deployments
  - nodes
  - pods
  position:
    top: 4
    left: 1
    height: 1
    width: 2
  refreshInterval: 300
  title: "Build System"
```
