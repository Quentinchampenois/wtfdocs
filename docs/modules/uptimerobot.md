# UptimeRobot

Connect to the [UptimeRobot](https://uptimerobot.com) API.

## Configuration

```yaml
uptimerobot:
  apiKey: "p0d13*********************************************c3"
  enabled: true
  offlineFirst: true
  position:
    top: 0
    left: 1
    height: 2
    width: 1
  refreshInterval: 3600
  uptimePeriods: 7-30
```
