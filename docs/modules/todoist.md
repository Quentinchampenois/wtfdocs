# Todoist

Displays all items on specified project.

## Configuration

```yaml
todoist:
  apiKey: "p0d13*********************************************c3"
  enabled: true
  position:
    top: 0
    left: 2
    height: 1
    width: 1
  projects:
    - 122247497
  refreshInterval: 3600
```
