# Buildkite

Connects to the [Buildkite](https://buildkite.com) REST API and displays status of branches for configured Pipelines.

The API token must have the `read_builds` permission.

## Configuration

```yaml
buildkite:
  apiKey: "p0d13*********************************************c3"
  enabled: true
  organizationSlug: "acme-corp"
  refreshInterval: 60
  position:
    top: 1
    left: 1
    height: 1
    width: 1
  pipelines:
    pipeline-slug-1:
      branches:
        - "master"
        - "stage"
    pipeline-slug-2:
      branches:
        - "master"
        - "production"
```
