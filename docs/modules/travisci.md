# TravicCI

Displays build information for your Travis CI account.

## Configuration

This module pulls information from <code>api.travis-ci.com</code> (when <code>pro: true</code>) or <code>api.travis-ci.org</code> (when <code>pro: false</code>).

```yaml
travisci:
  apiKey: "p0d13*********************************************c3"
  enabled: true
  compact: true
  limit: 8
  sort_by: "id:desc"
  position:
    top: 4
    left: 1
    height: 1
    width: 2
  pro: false
  refreshInterval: 900
```
