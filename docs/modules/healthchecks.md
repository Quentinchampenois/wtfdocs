# Healthchecks

Connect to the [Healthchecks](https://healthchecks.io) API.

## Configuration

```yaml
healthchecks:
  type: healthchecks
  apiKey: "XXXX"
  apiURL: "https://hc-ping.com/"
  tags:
    - backup
  position:
    top: 0
    left: 1
    height: 2
    width: 1
```
