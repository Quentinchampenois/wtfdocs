# New Relic

Connects to the New Relic API and displays the last n deploys of the
monitored applications: deploy ID, deploy time, and who deployed it.

## Configuration

```yaml
newrelic:
  apiKey: "p0d13*********************************************c3"
  applicationIDs:
    - 10549735
    - 499785
  deployCount: 6
  enabled: true
  position:
    top: 4
    left: 3
    height: 1
    width: 2
  refreshInterval: 900
```
